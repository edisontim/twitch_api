Gets a list of chat badges that can be used in chat for any channel. get-global-chat-badges

§Accessing the endpoint

§Request: GetGlobalChatBadgesRequest

To use this endpoint, construct a GetGlobalChatBadgesRequest with the GetGlobalChatBadgesRequest::new() method.

use twitch_api::helix::chat::get_global_chat_badges;
let request = get_global_chat_badges::GetGlobalChatBadgesRequest::new();

§Response: BadgeSet

use twitch_api::helix::{self, chat::get_global_chat_badges};
let request = get_global_chat_badges::GetGlobalChatBadgesRequest::new();
